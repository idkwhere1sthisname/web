Wii Shop Channel

The Wii Shop Channel lets you download software through the Internet that can be played on
the Wii Menu.

  • A Nintendo Network ID is required to use the Wii Shop Channel. For more information, see the "User Settings" section of the Wii U Electronic Manual ( ).
  • For more information about how to connect your system to the Internet, please review "Internet Settings" in the printed Wii U Operations Manual included with your system ( ).
  • Internet settings can be configured using System Settings, accessible from the Wii U Menu.

When the Wii Shop Channel Cannot be Used

In the situations listed below, the Wii Shop Channel cannot be used, even if you have registered a Nintendo Network ID and have configured a working Internet connection:

  • If the Country setting in System Settings within the Wii U Menu differs from the country setting for your Nintendo Network ID. (This can occur, for example, if you change the Country setting in System Settings after creating your Nintendo Network ID.)
  • If the Country setting in System Settings within the Wii U Menu is set to Bermuda or Malaysia.
