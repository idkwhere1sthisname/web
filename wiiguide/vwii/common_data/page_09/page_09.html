Wii Message Board

The Wii Message Board allows you
to view messages that are posted
on the calendar.

  • The calendar displays dates up through December 31, 2035.
  • The following features cannot be used:
  • Exchanging Wii messages with friends
  • Posting messages
  • Receiving notices over the Internet about events

Navigating the Message Board

Open message

Switch to the calendar screen

Move forward or back a day

Currently displayed date

Return to the Wii Menu

Today's Accomplishments

Each time you use software on the Wii Menu, the system records how long you spent playing it and then automatically sends you a message titled Today's Accomplishments.

  • The messages titled Today's Accomplishments cannot be deleted.

Selecting Dates and Viewing Messages

Select from the message board to display the calendar screen. The icon appears on days with messages.
To view the message board for a particular date,
just select the date.
