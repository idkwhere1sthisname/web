HOME Menu

The HOME Menu allows you to exit the software you're playing, configure your Wii Remote settings, and check the status of your Wii Remote controllers' batteries.

Displaying the HOME Menu

Press on the Wii Remote controller.

Navigating the HOME Menu

Return to the Wii Menu*

Display the instruction manual

(not shown for all software)

Return to previous screen

Return to the software's title screen*

Battery life of Wii Remote controllers

Display the Wii Remote Settings screen
  • IMPORTANT: Selecting the Wii Menu or Reset options will cause any unsaved information from the program you are currently running to be permanently lost.

Wii Remote Settings

This screen allows you to set the volume of the speaker,
turn the rumble feature on or off, and temporarily connect additional Wii Remote controllers.

Adjust the volume of the Wii Remote controller's built-in speaker*

Turn the rumble feature on or off*

Temporarily connect additional Wii Remote controllers†

Return to the HOME Menu
  • These settings apply to all Wii Remote controllers.
  • After you have connected all Wii Remote controllers that you will be using, press any button (except for the POWER Button) to return to the Wii Remote Settings screen. (You will automatically return to the Wii Remote Settings screen if you connect a fourth Wii Remote.)

Temporarily Connected Wii Remote Controllers

Wii Remote controllers will lose their connections under the following conditions and will need to be reconnected later:

  • When you return to the Wii U Menu
  • When you turn off the Wii U console
