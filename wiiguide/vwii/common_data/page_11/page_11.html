Screen Brightness

Pressing certain combinations of buttons on the Wii U GamePad adjusts the brightness and enables or disables power-saving mode for the Wii U GamePad screen.

Press and hold , and then press the following on :

Increase brightness

Decrease brightness

Disable power-saving mode

Enable power-saving mode
  • Screen brightness and power-saving mode settings cannot be set differently on the Wii Menu and Wii U Menu. Changes to settings made in one menu are applied to the other menu.

Battery Life

Battery life may differ depending on factors such as use of wireless communication, ambient temperature, and screen brightness. Enabling power-saving mode automatically adjusts the brightness of the screen depending on what is being displayed in order to conserve battery power.

  • The screen brightness is set to 4 by default.
  • If the screen brightness is set to 4 or 5, the screen will become brighter when using the AC adapter.
