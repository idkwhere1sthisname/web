SD Card Menu

The SD Card Menu lets you launch Wii software that is stored on SD (or SDHC) Memory Cards.

  • For more detailed information about the types of SD Cards that are supported, see "Using SD Cards" in the Wii U Electronic Manual ( ).

Displaying the SD Card Menu

1 Insert the SD Card that contains the software you want to use into the SD Card slot.

  • Check the orientation of the SD Card, and push it firmly into the slot until it clicks into place.

Troubleshooting the Error Message "This channel could not be launched"

The error message above means there is not enough free space in Wii system memory. Follow the instructions on the screen to manage your save data.

  • The SD Card Menu uses Wii system memory in order to launch channels.
