Playing Wii Discs

The basic procedures are described below.

  • To play a Wii U disc, you must return to the Wii U Menu (→).
  • Nintendo GameCube™ Game Discs are not supported.
Starting a Game

1 Press the EJECT Button to make sure there is no disc inserted in the console, and then insert the Wii disc.

2 Select the Disc Channel.

3 Select Start.

  • Review the instructions for the game you are playing for specific control options.
  • Features that use WiiConnect24 cannot be used.
  • Depending on the software, some features may be limited.

Exiting a Game

1 Press on the Wii Remote controller.

2 Select Wii Menu.

  • This option will cause any unsaved information from the program you are currently running to be permanently lost.

Ejecting a Disc

When you need to eject the disc, press the EJECT Button on
the console.
