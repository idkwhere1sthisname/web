Playing Wii Discs

To play Wii Game Discs, follow the steps given in the section below.

  • The Wii mini console does not connect to the Internet, and online features in games are not supported.
  • Nintendo GameCube™ Game Discs and accessories are not supported.
Launching Software Titles

1 Press the OPEN Button to open the cover. Load the Game Disc, label facing up, and press it into place.

2 Close the cover. If the console isn't already turned on, press the POWER Button.

3 Select the Disc Channel.

4 Select START.

  • Refer to the instruction booklet for the software title you are using for software-specific information.
  • WiiConnect24™ features are not supported.

Closing the Software

1 Press on the Wii Remote.

2 Select Wii MENU.

  • Please be careful when using this option, as any unsaved data will be lost.

Removing a Disc

To remove a disc, first press the POWER Button to turn off the console. Then press the OPEN Button and wait until the disc stops spinning before removing it carefully.
